# Home for You - Новогодний интернет-магазин

Проект новогоднего интернет-магазина с уютными товарами для дома.

## Функционал:
- Каталог из 8 товаров
- Фильтрация и сортировка
- Корзина с сохранением
- Адаптивный дизайн
- Новогодний счетчик

## Технологии:
- HTML5, CSS3, JavaScript
- LocalStorage для корзины
- Flexbox/Grid для верстки
- Font Awesome иконки

## Запуск:
Откройте `index.html` в браузере
